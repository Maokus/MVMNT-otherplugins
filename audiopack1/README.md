# Audio Pack 1

An SDK 2 MVMNT plugin generated with `create-mvmnt-plugin` and migrated from the original plugin.
The SDK dependency resolves to the sibling `MVMNT` checkout so it follows the current host API.

```sh
npm install
npm run typecheck
```

Preview or package it from a MVMNT checkout:

```sh
npm run dev-plugin -- /absolute/path/to/audiopack1
npm run build-plugin -- /absolute/path/to/audiopack1
```
